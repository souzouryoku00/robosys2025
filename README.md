# kadai01.mainコマンド
標準入力から受け取ったテキストに対して、文字数・単語数・行数・記号数などの統計情報を解析し、結果を出力するコマンド
文章の簡易的な解析やテキストの特徴量を確認する際に使用できます。

### 使い方
#### 1. リポジトリをクローン
```bash
git clone https://github.com/souzouryoku00/robosys2025
cd robosys2025
```
